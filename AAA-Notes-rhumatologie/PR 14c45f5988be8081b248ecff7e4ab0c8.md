# PR

De quoi meurent les gens atteints de PR : 

1. cardiovasculaires
2. infections pulmonaires
3. autres affections pulmonaires (dont les PIDs)

Risque infectieux dans la PR : 

- Le risque infectieux est directement corrélé à l’activité de la PR
- Les corticoïdes augmentent le risque infectieux mais pas les biothérapies enfaite
